## ✨ Features

- **Dark minimal design** with sage green and gold accents
- **Fully responsive** — works on mobile, tablet, and desktop
- **Smooth scroll animations** with Intersection Observer API
- **Active nav highlighting** as you scroll through sections
- **Zero dependencies** — no npm, no build step, no framework
- **Grain texture overlay** and animated radial glow effects
- **Google Fonts** — Cormorant Garamond, Cabinet Grotesk, Fira Code
